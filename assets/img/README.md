# Folder Logo SIWARGA

Upload 2 file logo di folder ini:

- **`logo_rt.jpg`** atau **`logo_rt.png`** — Logo RT (tampil di **KIRI** kop surat)
- **`logo_rw.png`** atau **`logo_rw.jpg`** — Logo RW (tampil di **KANAN** kop surat)

Rekomendasi: 300x300 px, format PNG transparan untuk hasil terbaik.

Jika salah satu/keduanya tidak ada, sistem akan skip tanpa error.

## Lokasi Logo Tampil:
1. **Login page** — ikon brand besar
2. **Cetak Surat** (kop surat resmi) — RT kiri, RW kanan
3. **Export PDF Laporan** — RT kiri, RW kanan, teks SIWARGA di tengah

## Cara Upload:

### XAMPP:
Copy kedua file logo ke `C:\xampp\htdocs\siwarga_php\assets\img\`

### cPanel:
File Manager → masuk `public_html/assets/img/` → Upload → pastikan nama persis `logo_rt.jpg` & `logo_rw.png`
